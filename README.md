# coffe

Site desenvolvido como atividade acadêmica utilizando:

- HTML
- CSS
- JavaScript
- Bootstrap 

## Funcionalidades
- Layout responsivo
- Seção hero
- Menu de navegação
- Botão interativo com JavaScript
- Publicação via GitHub Pages

## Processo de Desenvolvimento

O desenvolvimento iniciou-se com a definição da estrutura do site utilizando HTML semântico (header, nav, main, section, footer).
Em seguida, foi aplicada a estilização com CSS, priorizando organização, responsividade e uma identidade visual coerente com a proposta do café.

O Bootstrap foi utilizado para facilitar o layout responsivo e agilizar a criação de componentes.
A interatividade foi adicionada com JavaScript, principalmente para botões e ações no site.
Por fim, o projeto foi versionado com Git e publicado no GitHub Pages.

## Facilidades Durante o Desenvolvimento

Uso do Bootstrap, que agilizou a responsividade

HTML semântico facilitando a organização do código

## Dificuldades Encontradas

Organização inicial do layout e alinhamento de elementos

Ajustes de responsividade para telas menores

Configuração correta do repositório no GitHub

Entendimento do processo de merge e versionamento com Git
